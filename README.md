
#Tour-and-Tour Website

## Here's features included in this project

- Home Page
- Booking Page
- Show Differnt Types of Pakage 
- Show Servies of Tour Providing
- Gallery Page
- Review Page
- Contact Page
- Login and Register Page
- Sliding Search Bar  
- Go To Any Page Functionality

1. HTML
2. CSS
3. JavaScript
